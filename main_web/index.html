정교한 해킹 퍼즐

단순한 클릭이 아닌, 논리와 추론이 필요한 다양한 퍼즐과 난이도의 문제를 해결해야 합니다. 암호 해독, 네트워크 우회, 시스템 로그인 크랙 등 현실적인 해킹 과정을 게임으로 구현했습니다.

  • 퍼즐을 통해 해킹하는 기믹을 이용한 비밀번호 우회
  • 데이터 경로를 찾는 미로형 퍼즐
  • 시스템 파일을 조작하는 로직 문제
📜

어둠 속의 긴밀한 스토리

익명의 오퍼레이터(플레이어)는 다양한 클라이언트의 의뢰를 수행하며 거대한 디지털 범죄 조직과 비밀에 얽힌 스토리를 파헤치게 됩니다. 선택에 따라 스토리가 변화하는 깊이 있는 서사를 경험하세요.

  • 의뢰마다 달라지는 클라이언트와 타겟
  • 게임 내에서 실시간으로 업데이트되는 세션 로그
  • 진실에 다가갈수록 위험해지는 네트워크 환경
